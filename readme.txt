readme

This document serves as developer documentation for a road accident forecasting and data exploration project.


Usage: data_analyzer_R.R
	ui.R 			
	python predict_accident_randomforest.py 

1) data_analyzer_R.R
	R script to perform data munging and generate plots

2) ui.R and server.R
	R shiny user interface for road accident forecasting and data exploration.

3) functions_lib.R
	R script with forecasting algorithms

4) predict_accident_randomforest.py
	Function to load data and train random forest model of acccidents.

5) Data
	ACCIDENT.csv
	ATMOSPHERIC_COND.csv

	Accident data from https://www.data.vic.gov.au/data/dataset/crash-stats-data-extract

6) writeup_accident_prediction.doc

	Writeup on road accident forecasting and data exploration project.

7) installation_script.R
	R script to install all necessary packages.
